# 超短BTC信号系统 - TradingView Pine Script 使用说明

## 文件说明

- `ultra_short_btc.pine` - TradingView Pine Script 指标文件

## 安装方法

1. 打开 TradingView 网站（https://www.tradingview.com/）
2. 登录您的账号
3. 点击底部工具栏的 "Pine编辑器" 图标
4. 点击 "新建" 创建一个新脚本
5. 复制 `ultra_short_btc.pine` 文件中的全部内容
6. 粘贴到 Pine编辑器 中
7. 点击 "保存" 并给脚本命名（例如：超短BTC信号系统）
8. 点击 "添加到图表" 将指标添加到当前图表

## 功能说明

### 核心策略逻辑

1. **方向确认（1h级别）**
   - 使用EMA89、EMA144、EMA233判断多头趋势
   - 条件：EMA89 > EMA144 > EMA233

2. **入场信号（1-5min级别）**
   - 当价格接近布林带下轨时（在99%-100.5%之间）生成做多信号
   - 使用布林带（周期20，标准差2）

3. **止损止盈**
   - 止损：固定150点
   - 止盈：固定300点（2:1盈亏比）
   - 自动绘制止损和止盈线

### 参数设置

- **方向确认时间框架**：默认60（1小时），可自定义
- **入场信号时间框架**：默认5（5分钟），可选择1-5分钟
- **布林带周期**：默认20
- **布林带标准差**：默认2.0
- **止损点数**：默认150点
- **止盈倍数（盈亏比）**：默认2.0（即止损150点，止盈300点）
- **价格接近下轨容差**：默认1.005（100.5%）

### 显示选项

- 显示布林带：开启/关闭布林带显示
- 显示EMA：开启/关闭EMA显示
- 显示信号：开启/关闭信号标记
- 显示止损止盈线：开启/关闭止损止盈线

### 信息面板

指标会在图表右上角显示信息面板，包含：
- 方向时间框架
- 入场时间框架
- 多头趋势状态
- 接近下轨状态
- 信号状态
- 上次信号信息

### 警报设置

1. 在指标设置中，可以设置警报
2. 当新信号生成时，会触发警报
3. 警报消息包含：入场价格、止损价格、止盈价格

## 使用建议

1. **时间框架选择**
   - 建议在1分钟或5分钟图表上使用
   - 方向确认时间框架建议使用1小时

2. **信号确认**
   - 信号出现时，建议结合其他技术分析确认
   - 注意观察成交量变化
   - 考虑关键支撑位

3. **风险管理**
   - 严格执行150点止损
   - 达到300点止盈时及时平仓
   - 建议单次交易仓位不超过总资金的5%

4. **注意事项**
   - 该指标仅用于BTC交易对
   - 信号生成后需要手动执行交易
   - 市场波动大时注意控制风险

## 技术说明

- Pine Script版本：v5
- 兼容时间框架：所有时间框架
- 推荐时间框架：1分钟、5分钟图表
- 数据源：TradingView数据
